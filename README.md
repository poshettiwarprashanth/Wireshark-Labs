# Wireshark Labs

A practical, beginner-to-intermediate Wireshark lab series designed to build strong packet analysis and network troubleshooting skills through hands-on exercises.

---

## 📖 About

This repository contains structured Wireshark labs covering fundamental networking protocols, packet analysis, troubleshooting techniques, and security investigations.

Each lab focuses on a specific topic and includes step-by-step instructions, explanations, packet captures, screenshots, and key observations.

The goal is to understand how network communication works at the packet level rather than simply memorizing commands or filters.

---

## 🎯 Learning Objectives

By completing these labs, you will learn to:

- Capture network traffic
- Analyze packets
- Understand TCP/IP communication
- Inspect common network protocols
- Troubleshoot connectivity issues
- Identify abnormal network behavior
- Apply Wireshark filters effectively
- Perform basic network traffic analysis
